# 발달나침반 추가 기능 로드맵

## 구현 완료

- [x] 발달 체크리스트 (localStorage 기반, 로그인 불필요)

## 단기 (다음 우선순위)

### 활동 라이브러리
- 가정에서 할 수 있는 발달 촉진 놀이/활동 DB
- 연령별, 발달 영역별 필터
- 준비물, 소요시간, 난이도 표시
- 기존 가이드(guides) 구조 확장 가능

### 북마크 / 스크랩
- 로그인 사용자가 관심 콘텐츠 저장
- "나의 관심 주제" 대시보드
- 사용자 인증 시스템 선행 필요

### 용어 사전
- 논문/아티클에 나오는 전문 용어 풀이
- 아티클 내 용어 자동 링크 (tooltip)

## 중기

### 커뮤니티 / Q&A 게시판
- 부모들끼리 경험 공유, 질문/답변
- 카테고리별 필터, 좋아요, 댓글
- AI 상담 결과 익명 공유 기능
- **선행 필요: 공개 사용자 인증 시스템 (소셜 로그인 or 이메일 회원가입)**

### AI 상담 히스토리
- 이전 상담 결과 저장 및 재열람
- 시간에 따른 변화 추적
- 사용자 인증 필요

### 뉴스레터 / 월령별 알림
- 아이 생년월일 등록 시 해당 월령 맞춤 콘텐츠 발송
- "이번 달 우리 아이 발달 포인트" 형태
- 이메일 서비스 연동 필요 (Resend, SendGrid 등)

## 장기

### 전문가 칼럼 / 인터뷰
- 소아과 전문의, 언어치료사, 작업치료사 등 전문가 기고
- 논문과 다른 톤 — 더 실용적이고 경험 기반
- 새 콘텐츠 타입(experts) 또는 기존 topics 확장

### 발달 검사 기관 맵
- 지역별 발달 검사/치료 기관 정보
- 카카오맵/네이버맵 API 연동
- 기관 종류별 필터 (언어치료, 작업치료, 발달검사 등)

### 다국어 지원
- 다문화 가정을 위한 영어/베트남어/중국어 등
- next-intl 또는 next-i18next 도입

## 인프라 (여러 기능의 선행 조건)

### 공개 사용자 인증 시스템
- 현재: 관리자 전용 Supabase Auth
- 필요: 소셜 로그인 (카카오, 구글) 또는 이메일 회원가입
- 커뮤니티, 북마크, AI 상담 히스토리의 선행 조건
- Supabase Auth에 OAuth provider 추가로 구현 가능
